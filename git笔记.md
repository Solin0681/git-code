

# 	 git

## 步骤

1. ​	 git管理这个文件夹
2. ​	 进入文件夹 
3. ​	 初始化(相当于做标记)
4. ​	 管理 
5. ​	 生成版本

## 初始化(管理文件夹)	

git init

## 检查当前文件夹下的文件状态

git status    

## 管理文件

git add  文件名  

(git add  .   表示当前文件夹下的所有未管理的文件 开始进行管理)

## 生成版本

git  commit - m  '版本命名'
